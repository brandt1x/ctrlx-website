# Vision-X Tempo

Control-X branded GUI with red/black theme. Rebuilt from the NetX interface with simulated data.

## Launch

- **Double-click** `Launch Vision-X Tempo.bat` (first run will install dependencies automatically)
- Or run `npm install` then `npm start` in terminal

## Features

- **General** — Account info, expiration, version
- **Network** — Inbound/Outbound stats, Players, Sync, Stats
- **Vision-X** — Vision-X Shield and Vision-X settings
- **Debug** — Analysis, Controller State, Debug Out/In

## Simulate

Click **Simulate** in the top-right to start live-updating data. Status shows "Connected" in red when active.
